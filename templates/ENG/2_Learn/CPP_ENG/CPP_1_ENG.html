How to install C++?


  • Windows 10

    In order to have C++ working you have to download: Visual Studio Code on your Windows 10 Machine.

    In case if you have already a Windows 10 set up with Visual Studio Code Make sure to install C++ from the following link.


  • Linux

    In cases of Ubuntu Debian and common Linux Distributions you will have to open your terminal to enter the following command:

    sudo apt-get install g++ -y

    After hitting enter you will have to type your root password to proceed with the installation.


  • Text Editors or IDES?

    I recommend Microsoft Visual Studio, Geany or Sublime text editor however it's a good option to have something from Jetbrains but it's up to you!
