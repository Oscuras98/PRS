Como compilamos una programa de C++ en Linux usando g++?

  • Commando para tener un exe de 64 bits:

    g++ Cualquiera_Programa_DE_CPP_1.cpp -o Cualquiera_Programa_DE_CPP_1.exe -m64


    Commando para tener un exe de 32 bits:

    g++ Cualquiera_Programa_DE_CPP_2.cpp -o Cualquiera_Programa_DE_CPP_2.exe -m32
