Como crear comentarios en una programa escrita en c++?


  • Nuestro codigo de fuente original fue:

    #include <iostream>
int main()
{
    std::cout << "Hola Mundo!\n";
    return 0;
}

  • El comentario más simple es escribiendo //

    #include <iostream>
int main()
{
    // Nuestro primer comentario en c++!
    std::cout << "Hola Mundo!\n";// Este también es un comentario!
    return 0;//Nuestro tercero comentario!
    // Otro comentario!
}

    Al ejecutar nuestra programa podemos ver nuestro mensage hacia al mundo diciendo: "Hola Mundo".

    Por lo tanto los comentarios no están ejecutados!

  • Otro tipo de comentario en el ejemplo:

    #include <iostream>
int main()
{
    /* Este
    es
    un
    comentario
    en
    multiple
    lineas!*/
    std::cout << "Hola Mundo!\n";/* Nuestro segundo
    comentario en multiples
    Lineas!
    */
    return 0;/*Nuestro tercero comentario!*/
    /*Este tipo de comentario también puede estár escrita en una linea simple!*/
}

    Como la mencioné ahora los comentarios no están ejecutados al correr una programa! Solo sirven para dejar explicaciones en el fuente de codigo para los desarolladores!
