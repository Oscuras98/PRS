Hogyan telepíthető a C++?


  • Windows 10

    Ebben az esetben könnyű dolgunk van hiszen csak le kell tölteni a Microsoft oldaláról a Visual Studio Code Szöveg szerkesztő programot aminek a telepítőjében csak ki kell pipálni hogy szeretnénk telepíteni a C++ fejlesztő környezetet.
    Visual Studio Code

    Laikusoknak az alábbi linket ajánlanám vagy amennyiben már telepítve van a Microsoft Visual Studio Code akkor szintén az alábbi linket ajánlom!


  • Linux

    Általánosabb avagy elterjedtebb disztribuciók esetén a következő parancsot kell beírnunk a terminálba:

    sudo apt-get install g++ -y

    Amennyiben ez megtörtént a root jelszavunkat fogja kérni hogy folytassuk a telepítést.


  • Szövegszerkesztők vagy IDE mit válasszunk?

    Személy szerint a Microsoft Visual Studio Code-ot ajánlom vagy a Geany-t egyébb esetekben jól jöhet ha van egy Sublim Text editorunk is de ugyan úgy jó ötlet beszerezni egy eredti licesz kulcsot a JetBrains oldaláról!
